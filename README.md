# 5-Stage-Pipelined-RISC-V-Processor (RV321)
Designed and implemented a complete 32-bit 5-stage pipelined RISK-V (RV321) processor in Verilog HDL featuring hazard detection, data forwarding, branch handling, pipeline registers, instruction/data memory, and control logic. Succesfully synthesised, implemented, and timing-verified using Xilinx Vivado.
